This project is a 2D animated Solar System simulation developed using OpenGL and GLUT in C++. It visualizes the Sun, planets, moon, asteroid belts, galaxies, black holes, and other space objects with smooth animations and orbital movements. Each planet revolves around the Sun with different speeds to create a realistic effect. The project also includes background music, keyboard controls for pause/play and sound settings, and visual elements such as stars and meteoroids to create an interactive space environment. The purpose of this project is to demonstrate computer graphics concepts including transformations, animation, object rendering, and user interaction in OpenGL.
